Max Veytsman

I’m a human being living in Toronto. I like Clojure, Ruby, application security, and teaching other people about those things.

I hack things and build products at State Machinery, a computer security and software development consultancy I co-founded. We built gemcanary to help Ruby users monitor their dependencies for vulnerabilities, and we organize RubySec to provide security resources for the Ruby community.

In the summer of 2014, I went to Hacker School, an awesome retreat for programmers. During that time, I helped improve the state of Java security by injecting lolcats into software updates.

Also

I discovered a way to track the location of Tinder users by using trigonometry. Forbes and Businessweek wrote about it

I organize BSidesTO, a relaxed security conference in Toronto.
